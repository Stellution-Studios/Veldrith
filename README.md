__Veldrith__ is a high-performance, cross-platform graphics library for **.NET**, built on top of modern low-level APIs. It provides a clean, unified interface over **D3D12**, **Vulkan**, and **Metal**.

---

# ✨ About
Veldrith is designed to make modern GPU programming accessible and efficient within the .NET ecosystem. Whether you're building a game engine, a real-time renderer, or a visualization tool, Veldrith gives you the power of low-level graphics APIs without the boilerplate.

- 🔥 **High Performance** — Built on D3D12, Vulkan, and Metal for maximum throughput and minimal overhead.
- 🌐 **Cross-Platform** — Write once, run on Windows, Linux, macOS, and beyond.
- 🧩 **Unified API** — One clean interface across all backends, no backend-specific code needed.
- ⚡ **Modern .NET** — Fully compatible with .NET 10+, designed with performance-first principles in mind.

# 🪙 Installation - [Nuget](https://www.nuget.org/packages/Veldrith)

```
dotnet add package Veldrith --version [VERSION]
```

# 📖 Build from Source

> 1. Clone this repository.
> 2. Add `Veldrith.csproj` as a reference to your project.
